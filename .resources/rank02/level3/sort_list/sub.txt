Assignment name  : sort_list
Expected files   : sort_list.c
Allowed functions:
--------------------------------------------------------------------------------
以下の関数を作成してください:

t_list	*sort_list(t_list* lst, int (*cmp)(int, int));

この関数は、パラメータとして与えられたリストをソートし、関数ポインタcmpを使用して
適用する順序を選択し、ソートされたリストの最初の要素へのポインタを返す必要があります。

重複は残す必要があります。
入力は常に一貫性があります。

提供されるファイルlist.hに記述されている型t_listを使用する必要があります。
そのファイルをインクルードする必要がありますが(#include "list.h")、提出する必要は
ありません。課題のコンパイルには独自のものを使用します。

cmpとして渡される関数は、aとbが正しい順序である場合は0とは異なる値を返し、
そうでない場合は0を返します。

例えば、cmpとして使用される以下の関数は、リストを昇順にソートします:

int ascending(int a, int b)
{
	return (a <= b);
}
LIST.H FILE:

typedef struct s_list t_list;

struct s_list {
    int data;
    t_list *next;
};
このシェルでは、自分のLIST.Hファイルでテストする必要があります。
