Assignment name  : ft_list_remove_if
Expected files   : ft_list_remove_if.c
Allowed functions: free
--------------------------------------------------------------------------------

渡されたリストから、データが参照データと"等しい"すべての要素を削除する
ft_list_remove_ifという関数を作成してください。

関数は以下のように宣言されます:

void ft_list_remove_if(t_list **begin_list, void *data_ref, int (*cmp)());

cmpは2つのvoid*を受け取り、両方のパラメータが等しい場合に0を返します。

以下の内容を含むft_list.hファイルを使用する必要があります:

$>cat ft_list.h
typedef struct      s_list
{
    struct s_list   *next;
    void            *data;
}                   t_list;
