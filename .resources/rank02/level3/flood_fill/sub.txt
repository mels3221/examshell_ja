Assignment name  : flood_fill
Expected files   : flood_fill.c
Allowed functions: -
--------------------------------------------------------------------------------

char **を2次元のchar配列として受け取り、t_pointをこの配列の次元として、t_pointを開始点
として受け取る関数を作成してください。

与えられた'begin' t_pointから開始して、この関数は内部の文字を文字'F'に置き換えることで
領域全体を埋めます。ゾーンとは、水平および垂直に他の文字または配列の境界によって区切られた
同じ文字のグループです。

flood_fill関数は斜めには埋めません。

flood_fill関数は以下のようにプロトタイプ宣言されます:
  void  flood_fill(char **tab, t_point size, t_point begin);

t_point構造体は以下のようにプロトタイプ宣言されます: (flood_fill.cに入れてください)

  typedef struct  s_point
  {
    int           x;
    int           y;
  }               t_point;

Example:

$> cat test.c
#include <stdlib.h>
#include <stdio.h>

char** make_area(char** zone, t_point size)
{
	char** new;

	new = malloc(sizeof(char*) * size.y);
	for (int i = 0; i < size.y; ++i)
	{
		new[i] = malloc(size.x + 1);
		for (int j = 0; j < size.x; ++j)
			new[i][j] = zone[i][j];
		new[i][size.x] = '\0';
	}

	return new;
}

int main(void)
{
	t_point size = {8, 5};
	char *zone[] = {
		"11111111",
		"10001001",
		"10010001",
		"10110001",
		"11100001",
	};

	char**  area = make_area(zone, size);
	for (int i = 0; i < size.y; ++i)
		printf("%s\n", area[i]);
	printf("\n");

	t_point begin = {7, 4};
	flood_fill(area, size, begin);
	for (int i = 0; i < size.y; ++i)
		printf("%s\n", area[i]);
	return (0);
}
