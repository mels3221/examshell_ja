Assignment name  : do_op
Expected files   : do_op.c
Allowed functions: atoi, printf, write
--------------------------------------------------------------------------------

3つの文字列を受け取るプログラムを作成してください:
- 1番目と3番目の文字列は、int型に収まる10進数の符号付き整数の表現です。
- 2番目の文字列は、+ - * / % から選択された算術演算子です。

プログラムは要求された算術演算の結果を表示し、その後に改行を出力する必要があります。
パラメータの数が3でない場合、プログラムは単に改行を表示します。

文字列には間違いや余分な文字がないと仮定できます。入力または出力の負の数には、
1つだけ先頭に'-'が付きます。演算の結果はint型に収まります。

Examples:

$> ./do_op "123" "*" 456 | cat -e
56088$
$> ./do_op "9828" "/" 234 | cat -e
42$
$> ./do_op "1" "+" "-43" | cat -e
-42$
$> ./do_op | cat -e
$
