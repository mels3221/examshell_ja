Assignment name  : ft_list_size
Expected files   : ft_list_size.c
Allowed functions:
--------------------------------------------------------------------------------

渡された連結リスト内の要素数を返す関数を作成してください。

関数は以下のように宣言する必要があります:

int	ft_list_size(t_list *begin_list);

プログラムft_list_size.cでは以下の構造体を使用する必要があります:

typedef struct    s_list
{
    struct s_list *next;
    void          *data;
}                 t_list;
