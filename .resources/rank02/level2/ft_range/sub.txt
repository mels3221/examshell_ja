Assignment name  : ft_range
Expected files   : ft_range.c
Allowed functions: malloc
--------------------------------------------------------------------------------

以下の関数を作成してください:

int     *ft_range(int start, int end);

この関数は(malloc()を使用して)整数の配列を割り当て、startで始まりendで終わる連続した値で
埋める必要があります(startとendを含みます!)。その後、配列の最初の値へのポインタを
返します。

Examples:

- (1, 3)の場合、1, 2, 3を含む配列を返します。
- (-1, 2)の場合、-1, 0, 1, 2を含む配列を返します。
- (0, 0)の場合、0を含む配列を返します。
- (0, -3)の場合、0, -1, -2, -3を含む配列を返します。
