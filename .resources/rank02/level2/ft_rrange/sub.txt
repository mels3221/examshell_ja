Assignment name  : ft_rrange
Expected files   : ft_rrange.c
Allowed functions: malloc
--------------------------------------------------------------------------------

以下の関数を作成してください:

int     *ft_rrange(int start, int end);

この関数は(malloc()を使用して)整数の配列を割り当て、endで始まりstartで終わる連続した値で
埋める必要があります(startとendを含みます!)。その後、配列の最初の値へのポインタを
返します。

Examples:

- (1, 3)の場合、3, 2, 1を含む配列を返します。
- (-1, 2)の場合、2, 1, 0, -1を含む配列を返します。
- (0, 0)の場合、0を含む配列を返します。
- (0, -3)の場合、-3, -2, -1, 0を含む配列を返します。
