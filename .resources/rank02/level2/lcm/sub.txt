Assignment name  : lcm
Expected files   : lcm.c
Allowed functions:
--------------------------------------------------------------------------------

2つのunsigned intをパラメータとして受け取り、それらのパラメータの最小公倍数(LCM)を
計算して返す関数を作成してください。

LCM(最小公倍数)とは、2つのゼロでない整数の両方で割り切れる最小の正の整数です。

LCMは2つの方法で計算できます:

- 0以外の公倍数が見つかるまで、各整数のすべての倍数を計算することができます。

- これら2つの整数のHCF(最大公約数)を使用し、以下のように計算することができます:

	LCM(x, y) = | x * y | / HCF(x, y)

  | x * y | は「xとyの積の絶対値」を意味します

少なくとも1つの整数がnullの場合、LCMは0に等しくなります。

関数は以下のようにプロトタイプ宣言する必要があります:

  unsigned int    lcm(unsigned int a, unsigned int b);
