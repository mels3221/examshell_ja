Assignment name  : ft_atoi_base
Expected files   : ft_atoi_base.c
Allowed functions: None
--------------------------------------------------------------------------------

文字列引数str(基数N <= 16)を整数(10進数)に変換して返す関数を作成してください。

入力で認識される文字は: 0123456789abcdef
これらは、もちろん要求された基数に応じて切り詰められます。例えば、基数4は"0123"を認識し、
基数16は"0123456789abcdef"を認識します。

大文字も認識される必要があります: "12fdb3"は"12FDB3"と同じです。

マイナス記号('-')は、文字列の最初の文字である場合にのみ解釈されます。

関数は以下のように宣言する必要があります:

int	ft_atoi_base(const char *str, int str_base);
