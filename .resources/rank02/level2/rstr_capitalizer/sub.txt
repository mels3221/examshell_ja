Assignment name  : rstr_capitalizer
Expected files   : rstr_capitalizer.c
Allowed functions: write
--------------------------------------------------------------------------------

1つ以上の文字列を受け取り、各引数について、各単語の文字である最後の文字を大文字にし、
残りを小文字にし、その結果を表示し、その後に\nを出力するプログラムを作成してください。

単語とは、スペースやタブで区切られた文字列の一部分、または文字列の開始/終了によって
区切られた部分のことです。単語が1文字のみの場合、それは大文字にする必要があります。

文字とは、[a-zA-Z]のセット内の文字です。

パラメータがない場合、\nを表示してください。

Examples:

$> ./rstr_capitalizer | cat -e
$
$> ./rstr_capitalizer "a FiRSt LiTTlE TESt" | cat -e
A firsT littlE tesT$
$> ./rstr_capitalizer "SecONd teST A LITtle BiT   Moar comPLEX" "   But... This iS not THAT COMPLEX" "     Okay, this is the last 1239809147801 but not    the least    t" | cat -e
seconD tesT A littlE biT   moaR compleX$
   but... thiS iS noT thaT compleX$
     okay, thiS iS thE lasT 1239809147801 buT noT    thE leasT    T$
$>
